# OpenAI API Reference in Markdown

The complete [OpenAI API reference](https://platform.openai.com/docs/api-reference) converted to Markdown format.

## Why?

Include this repository in your project to give your coding agent access to the official OpenAI API documentation. This enables accurate code review and validation when working with OpenAI APIs.

*2026-01-14*

This repository is an exact copy of the official documentation as of the date above.
